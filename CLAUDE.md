# NoteMath

## Purpose
A cute, interactive calculator website where users type words and messages instead of numbers. Users can create equations like "me + you = 💕" and send them to friends via text message.

## Target Audience
Gen Z and college students. Young women aged 18-25.

## Vibe & Feeling
Cute, wholesome, girlie, playful, fun. Think Y2K meets modern pastel aesthetic.

## Colours
- Background: Off-white (#fafaf8)
- Calculator body: Mix of pinks — baby pink, hot pink accents, dusty rose
- Buttons: Various pink tones with soft shadows
- Text on buttons: White or dark pink
- Equals button: Hot pink, bold

## Typography
- Friendly, rounded font — use Google Fonts (Nunito or Poppins)
- Playful but readable
- Display screen font, pixelated (Bitcount Prop Double)

## Key Features
- Calculator centred on page
- Letter buttons instead of number buttons
- Maths symbols: +, -, x, ÷, =
- Display screen shows the message being typed
- Special buttons: space bar, heart emoji 💕, clear button
- Share via text message button after pressing equals
- Button press animations — bouncy, satisfying
- Mobile first — must work perfectly on phones

## Rules
- Always mobile first
- Keep it simple and intuitive — no instructions needed
- Never use harsh colours or sharp corners
- Buttons must always be rounded
- The whole experience should feel like a gift to someone
## Design Style
- Retro Windows calculator aesthetic
- Tall rectangle, taller than wide
- Black title bar at top with "NoteMath" and checkbox icon
- Light grey calculator body
- White buttons with dark borders and slight 3D pressed effect
- Monospace system font
- Hot pink equals button is the only colour accent
- No gradients, no glow, no shadows behind calculator
- Letters A-Z replace numbers
